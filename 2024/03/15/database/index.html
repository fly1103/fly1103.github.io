    Oceanbase相关

    Mysql模式

    1.连接租户

    mysql -h 172.20.249.50 -u root@sys#obdemo -P 2883 -p password -c -A oceanbase

    -h:ip地址
    -u：提供租户的连接账户，提供租户的连接账户，格式有两种：用户名 @ 租户名 # 集群名 或者 集群名 : 租户名 : 用户名 。MySQL 租户的管理员用户名默认是 root。
    -P:提供 OceanBase 数据库连接端口，也就是 OBProxy 的监听端口，默认是 2883 ，可以自定义。
    -p:提供账户密码，为了安全可以不提供，改为在后面提示符下输入，密码文本不可见
    -c：表示在 MySQL 运行环境中不要忽略注释。
    -A：表示在 MySQL 连接数据库时不自动获取统计信息
    oceanbase ：访问的数据库名，可以改为业务数据库。

    2.查看数据库

    show databases;

    3.进入数据库

    use database_name;

    Oracle模式

    1.连接租户

    obclient -h 172.20.249.50 -u root@sys#obdemo -P 2883 -p password -c -A oceanbase

    DRS相关

    1.修改云下drs node最大值

    vim /drs/package/安装时间/conf/node-agent-config.properties
    修改cluster.node.maxJobCount=10
    ps -ef | grep node-agent
    kill pid
    等待1分钟自动拉起

    可以su - omm
    查看该视图，可以看到相关信息
    select * from core.node_to_job_mgr_heartbeat;

    2.创建心跳表

    create table 数据库.drs_heartbeat(col_time timestamp(0));
    insert into 数据库.drs_heartbeat values(now());
    select DBE_TASK.SUBMIT(‘update 数据库.drs_heartbeat set col_time=now() limit 1;’,sysdate,‘sysdate+1/24’);
    alter table 数据库.drs_heartbeat owner to drsuser;

    alter table 数据库.drs_heartbeat replica identity full;

    UGO相关

    1.安装UGO

    1.创建ugo用户

    useradd ugo

    password ugo

    password:Picc@2024ugo

    2.获取 Web 认证文件

    keytool -genkey -alias UGOWebKeystoreCRT -validity 3650 -keyalg RSA -keystore “/home/ugo/UGOWebKeystore” -keysize 2048 -storetype PKCS12

    chown ugo:ugo /home/ugo/UGOWebKeystore

    申请证书密码：Gauss@123

    3.安装 ThirdPartySoftware 依赖

    tar -zxvf ThirdPartySoftware.tar.gz

    sh install.sh

    4.安装 UGO

    chown -R ugo:ugo UGO.tar.gz

    su - ugo 进入ugo用户

    tar -zxvf UGO.tar.gz

    cd UGO/

    vi install-default.conf

    common.install.path /gaussdb/ugoserver/ UGO安装路径
    common.log.path /gaussdb/ugoserver/logs/ UGO通用日志路径

    data路径为 /gaussdb/ugoserver/ db/data

    python3 install.py install --install-repo-db --license /home/ugo/LICUGO_2_20240305O40Z5L.xml --web-cert /home/ugo/UGOWebKeystore

    dbuser:ugodb

    password:Gauss@123

    UGO的用户名默认为admin，密码为admin密码，即安装时为IAM User设置的密码。Gauss@123
